Projet Objectif Action

Créer des tâches rapidement si elles ne sont pas présentent sur l'outil Pivotal Tracker.
Pour ce faire il suffit de remplir les heures de chaques taches séparées par un "+".
Exemple : 2+2+0+1+1+9+9+6

Si la tâche est présente dans Pivotal Tracker, après avoir choisi le bon projet, appuyez sur "Rechercher".
En fonction du nombre de stories présentent dans le projet, le chargement peut prendre entre 0 et 30+ secondes.
Une fois le chargement fini, vous pouvez utiliser la barre de recherche pour afficher rapidement une story qui vous intéresse.
Pour choisir les tâches qui vous intéressent, changez "Non" par le bon nombre d'heure.
Une fois toutes les tâches choisies pour un projet, cliquer sur "Attribuer".
